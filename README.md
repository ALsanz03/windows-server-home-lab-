# Windows Server 2025 Home Lab

## Objective

Build a Windows Server environment using Hyper-V to practice system administration and networking skills.

## Skills Demonstrated

- Hyper-V
- Windows Server 2025
- Active Directory
- DNS
- DHCP
- User Account Management
- Troubleshooting

## Environment

Host:
Windows 11

Virtual Machines:
- Windows Server 2025
- Windows 11 Client

## Challenges

### PXE Boot Error

Issue:
VM displayed "Start PXE over IPv4"

Resolution:
Verified ISO attachment and adjusted VM boot order.
