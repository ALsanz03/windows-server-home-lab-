# Windows Server 2025 Home Lab

## Objective

Designed and deployed a Windows Server 2025 Active Directory homelab using Hyper-V to simulate a small business network environment. Configured Active Directory Domain Services (AD DS) and DNS, promoted the server to a Domain Controller, and created domain users and groups for centralized identity management.

Implemented Organizational Units (OUs), security groups, and user accounts following standard Active Directory administration practices. Configured and tested domain authentication, DNS resolution, and user management within the domain environment.

Documented system configuration, troubleshooting procedures, and deployment steps, including DNS configuration, domain controller promotion, and Active Directory administration. The lab was built to develop hands-on experience with Windows Server technologies commonly used in enterprise IT environments.

Technologies Used: Windows Server 2025, Hyper-V, Active Directory Domain Services (AD DS), DNS, Group Policy, Windows 11, PowerShell.
